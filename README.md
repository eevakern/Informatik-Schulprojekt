Informatik Projekt: Filmdatenbank

Ich habe meine Lieblingsfilme ausgewählt, weil ich gedacht habe, wäre es nicht einfach,
auf einen Blick meine Lieblingsfilme zu sehen? um diese sich dann natürlich nochmal angucken
zu können, wenn man nicht weiß, was man gucken soll. 

Live-Webseite: https://eevakern.github.io/Informatik-Schulprojekt/index.html

So habe ich mein Projekt umgesetzt:
1. Datenbank und Design:
Zuerst habe ich die Datenbank für die Filme mit SQLite erstellt und als Datei
gespeichert (Daten_Eva.db) und nochmal in (SQLite.json) . Danach habe ich die Webseite mit HTML und CSS gebaut.
Mir war wichtig, dass das Design modern aussieht (dunkler Hintergrund und rote Akzente, so ein bisschen, wie im Kino),
damit es gut zu dem Thema Filme passt.

3. Die Suche nach einem Server:
Ich wollte, dass die Webseite nicht nur auf meinem Laptop liegt, sondern dass
man sie wie eine echte Internetseite überall aufrufen kann. Deshalb habe ich nach
einer kostenlosen Lösung gesucht, um die Seite online zu stellen („Hosting“).
So bin ich auf GitHub gekommen.

5. Hochladen und Veröffentlichen:
Ich habe mir bei GitHub einen Account erstellt und dort meine 5 fertigen Dateien
hochgeladen. GitHub bietet eine Funktion an, die „GitHub Pages“ heißt. Damit wird
aus den hochgeladenen Dateien automatisch eine richtige Webseite gemacht.

7. Wie es funktioniert:
Das Ganze ist jetzt so eingestellt, dass mein JavaScript-Code die Datenbank-Datei
direkt vom GitHub-Server lädt, sobald man auf den Button „Filme anzeigen“ klickt.
Die Daten werden dann automatisch in die Tabelle auf der Seite geschrieben. So ist alles
online und funktioniert ohne Probleme!
